Etiquetas Semanticas Estructurales

DIV: Es una etiqueta contenedora que no tiene semantica
HEADER: cabecera de un sitio web o una seccion
MAIN: define la seccion principal del document, solo puede existir una por documento
FOOTER: pié de pagina de una sitio web o una seccion
NAV: representa una menu de navegacion
ARTICLE: representa una seccion de autocontenido (que por si sola se explica)
ASIDE: representa contenido complementario o secundario
SECTION: representa una seccion de contenido generico
style="background-color:aquamarine" ADDRESS: representa una direccion fisica
